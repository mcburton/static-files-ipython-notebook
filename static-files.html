Playing with static files in the iPython Notebook

So there is a neat and only marginally documented feature in the iPython notebook. Since the notebook is a web server, it can serve up files beyond those in your notebook. Specifically, it can serve up files the Notebook's directory.

You can view this notebook on the nbviewer here and I've uploaded the various files to a github repository here

NOTE: If you are viewing the static version of this Notebook none of the links or iframes will work. I have included screenshots of the iframes for your convenience, but the FileLinks() and markdown links won't work.

%%writefile jsonfoo.html
<html>
    <head>
    <title>Some HTML file</title>
    </head>
    <script src="//cdnjs.cloudflare.com/ajax/libs/d3/3.2.2/d3.v3.min.js"></script>
    <script>
    d3.json("foo.json", function(data) {
        d3.select("ul").selectAll("li")
            .data(data).enter()
                .append("li")
                    .text(function(d) { return d; });
    });
    </script>
    <body>
        <ul>
    </body>
</html>

Based upon this notebook on iPython's Rich Display System we can do a few more fun things with the static file serving capabilties.

For example, we can use the %%HTML magic to load up files/jsonfoo.html in an iframe.

Pretty neat. There are some gotchas however. When I changed foo.json on disk, I had to open up foo.json in a new tab and force a refresh because the browser cache was stale. Something to keep in mind if you aren't seeing changes being reflected.
